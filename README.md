# Monthly Sales Performance Report
### Excel Basics & Data Management Project | TuteDude
## Overview
A structured Excel workbook built to demonstrate core Microsoft Excel skills including data entry, formula-based calculations, professional formatting, and data manipulation techniques.
## Workbook Structure
The workbook contains 3 sheets:
- **Sales Data** - main dataset with formulas and summary calculations
- **Copied Data** - dataset pasted using Paste Special (Values Only)
- **Transposed Data** - rows and columns flipped using transpose
## What I Did
**Part 1 - Data Entry**
Created a Sales Data sheet with 10 salesperson entries across columns - Name, Region, Product, Units Sold, Price per Unit, Total Sales, Incentive, and Incentive Rate.
**Part 2 - Formulas**
- Total Sales = Units Sold x Price per Unit
- Incentive = IF Total Sales > 50,000 - 10%, Otherwise - 5%
- Incentive Rate shown as a seperate percentage column
**Part 3 - Formatting**
Applied ₹ currency format, bold headers, background colours, borders on all cells, centre-aligned headings, and adjusted row height and column widths.
**Part 4 - Cut, Copy & Paste Special**
- Copied dataset to a new sheet using Paste Special - Values Only (removes formulas, retains computed values)
- Created a Transposed sheet where rows become columns and columns become rows
